# 鱼盆模型量化分析系统

基于鱼盆模型的多Agent量化交易分析系统，自动采集A股宽基/窄基/个股、美股、日本市场、港股市场和大宗商品数据，分析趋势强度并生成交易信号。

## 功能特性

- **多数据源采集**：支持新浪财经、中证CSIndex、东方财富、腾讯财经、Baostock等数据源，各品类自动切换备用源，兼容宽基指数、窄基/行业指数（含华证代码）及A股个股
- **PE估值数据**：集成蛋卷基金PE/PB百分位数据，辅助判断指数估值水平（1小时缓存）
- **多Agent架构**：数据采集、数据分析、信号生成、报告生成四个Agent协同工作
- **鱼盆模型策略**：基于20日均线判断强弱状态，结合偏离度生成5级交易信号
- **状态转变标记**：当指数YES/NO状态在当日发生转变时，HTML报告中会以⚡标记高亮提醒
- **数据时效提示**：各分组数据更新时间标签自动染色——当日数据显示绿色，非当日数据显示橙色警告
- **分组展示**：趋势强度排名按市场分组（🇨🇳 A股 / 🇺🇸 美股 / 🇯🇵 日本市场 / 🇭🇰 港股市场 / 🏅 大宗商品 / 🎯 A股窄基行业趋势 / 📌 A股个股趋势），个股不参与排名及整体趋势统计
- **多格式报告**：支持JSON、Markdown、HTML三种输出格式；可通过开关控制"数据源/PE百分位"列是否展示
- **定时任务**：按配置时间执行分析（默认交易日 19:00，每周一到周五）

## 数据源说明

各市场数据源及优先级如下：

| 市场 | 主数据源 | 备用数据源 |
|------|---------|----------|
| A股宽基 | 新浪财经 | 中证CSIndex → Baostock → 东方财富 → 腾讯财经 |
| A股窄基 | 腾讯财经 | 中证CSIndex |
| A股个股 | 新浪财经 | — |
| 美股 | 新浪美股指数 | — |
| 日本市场 | 新浪全球指数 | — |
| 港股市场 | 新浪港股指数 | 东方财富 |
| 大宗商品 | 外盘商品期货 | — |

> 各数据源更新时间存在差异，建议收盘后晚间执行。港股数据在东方财富接口不可用时会自动降级为新浪源，可能落后一个交易日（报告中以橙色标注）。

## 使用方法

### 快速运行

```bash
./run.sh
```

脚本会自动创建虚拟环境（若不存在）并安装依赖，无需手动激活。

### Windows 一键运行

```powershell
.\run_windows.ps1
```

### 手动运行

```bash
source .venv/bin/activate
cd src
python main.py
```

### 查看报告

报告文件保存在 `yupen/outputs/` 目录下：

- `latest_report.html` - 最新HTML报告（可在浏览器中打开）
- `latest_report.json` - 最新JSON报告
- `latest_report.md` - 最新Markdown报告
- `yupen_report_YYYYMMDD_HHMMSS.*` - 带时间戳的历史报告

### 添加自选标的

编辑 `yupen/src/config.py` 中的 `INDICES` 列表，按以下格式添加：

```python
# A股宽基指数
{"name": "名称", "code": "代码", "market": "sh"},   # 上交所
{"name": "名称", "code": "代码", "market": "sz"},   # 深交所

# 窄基/行业指数（含华证等非标准代码）
{"name": "名称", "code": "代码", "market": "narrow"},

# A股个股
{"name": "名称", "code": "代码", "market": "stock"},

# 美股
{"name": "名称", "code": "代码", "market": "us"},

# 港股
{"name": "名称", "code": "代码", "market": "hk"},
```

### 其他配置

| 配置项 | 位置 | 说明 |
|--------|------|------|
| 交易时间 | `REPORT_CONFIG["交易时间"]` | 定时任务执行时间，默认 19:00 |
| 数据源/PE列 | `REPORT_CONFIG["显示数据源PE指标"]` | True 显示，False 隐藏 |

## 缓存机制

- **蛋卷基金PE数据**：缓存1小时，避免频繁请求
- 各数据源采集结果在单次运行内复用，多次运行间不缓存

## 鱼盆模型说明

鱼盆模型是一种趋势跟随策略：

- **YES状态**：价格站上20日均线，看多
- **NO状态**：价格跌破20日均线，看空
- **偏离度**：当前价格与MA20的偏离百分比，反映趋势强弱
- **涨跌幅**：当日相对前日的涨跌百分比
- **PE百分位**：当前PE值在历史区间中的百分位排名（来自蛋卷基金），辅助判断估值高低
- **趋势强度排名**：按偏离度从大到小排序，仅含指数（个股不参与）

### 交易信号等级

| 偏离度范围 | 信号 | 含义 |
|-----------|------|------|
| ≥ 3% | 🟢 强力买入 | 趋势强劲，积极持有/买入 |
| 1% ~ 3% | 🟢 买入 | 持有为主，可适当加仓 |
| 0% ~ 1% | 🟡 观望 | 谨慎持有，等待明确信号 |
| -1% ~ 0% | 🟠 谨慎 | 考虑减仓，控制风险 |
| < -1% | 🔴 卖出 | 及时止损或减仓 |
