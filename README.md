## 项目列表

### yupen

**鱼盆模型量化分析系统** - 基于鱼盆模型的多Agent量化交易分析系统

- 多数据源采集（新浪财经、中证CSIndex、东方财富、腾讯财经、Baostock）
- 多Agent架构协同工作
- 基于20日均线的指数趋势分析
- 自动生成JSON、Markdown、HTML报告
- 每日15:30定时执行

详情见 [yupen/README.md](yupen/README.md)
