# 质量检查 Prompt

## 适用场景

当我对文章基本满意后，使用这个 Prompt 做最后一轮检查和优化。

这一轮不是重新写一篇文章，而是在不改变核心立场的前提下，帮助我提高文章质量。

## Prompt

```text
我对这版文章基本满意了。现在请做最后一轮质量检查，不要大改文章立场，也不要重新写成另一篇文章。

请从三个角度检查：

1. 反方审稿

请检查：

- 哪些观点可能被读者反驳？
- 有没有过度绝对的判断？
- 有没有逻辑跳跃？
- 有没有需要补充边界的地方？
- 有没有容易被误解的表达？
- 是否有必要加入反例或限定条件？

2. 事实风险扫描

请检查：

- 哪些句子涉及事实、数据、公司、产品、人物、时间、行业案例、报告或政策？
- 哪些内容需要查证？
- 哪些说法可能凭印象写得太死？
- 哪些地方应该改成更稳妥的表达？
- 哪些观点应该明确为作者判断，而不是事实结论？

3. 去 AI 腔优化

请检查：

- 哪些地方像 AI 写的？
- 哪些地方像行业报告或咨询白皮书？
- 哪些表达太空泛、太顺滑、太正确但没有作者感？
- 哪些句子可以更自然、更有真实思考感？
- 是否有过度排比、过度总结或套话？

请先输出检查意见。

然后在不改变文章核心立场的前提下，给出一版优化后的稿子。

优化时请注意：

- 保留我的核心观点；
- 保留文章原有判断；
- 不要把文章改得过于中庸；
- 不要为了稳妥而牺牲表达力度；
- 不确定事实要改得更谨慎；
- 语言要更自然，更像真实作者。
```
