# 观点拓展 Prompt

## 适用场景

当我有一个文章主题和大致想法，但还不确定文章主线、核心概念或可展开角度时，使用这个 Prompt。

这个阶段不要写正文。

## Prompt

```text
我想写一篇观点型文章。

主题：

我的想法：

先不要写正文。请先帮我做一轮观点拓展。

请输出以下内容：

1. 你理解到的核心观点

请复述你理解到的文章核心观点，确认你没有理解偏。

2. 可以提炼出的核心概念

请从我的想法中提炼几个适合支撑文章的核心概念。

这些概念要能帮助文章形成主线，而不是为了制造术语。

3. 我可能没想到的补充角度

请补充一些我可能没想到、但和主题高度相关的角度。

这些角度可以包括新的概念、反例、边界、类比、结构化框架或延伸判断。

4. 拓展内容分类

请把拓展内容分成三类：

- 核心拓展：强相关，建议写进正文；
- 辅助拓展：有启发，可选择性使用；
- 发散拓展：有意思，但可能偏题，暂时作为备用。

5. 事实风险提示

请指出文章里可能涉及哪些事实类内容需要查证，比如公司、人物、产品、数据、发布时间、行业案例、报告、政策等。

6. 推荐文章主线

请给出你建议的文章主线或结构。

注意：

- 你可以拓展我的观点，但不能替换我的核心立场。
- 不要把文章方向带散。
- 不要为了显得全面而加入太多无关内容。
- 先不要写正文，等我确认哪些拓展保留后再写初稿。
```
